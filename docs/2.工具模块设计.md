# GraphAgent 工具模块设计文档

## 1. 概述

为了支持大规模工具的定义、管理和扩展，GraphAgent 实现了统一的工具模块架构,提供了工具注册、发现、管理和调用的完整生命周期支持。

## 2. 目录结构

```
src/graph_agent/
├── tools/
│   ├── __init__.py          # 公共接口导出
│   ├── base.py              # 核心基础类（ToolCenter、AgentTool、装饰器）
│   ├── time_tools.py        # 时间相关工具
│   ├── calc_tools.py        # 计算相关工具
│   └── ...                  # 更多工具文件
└── graph.py                 # Agent 图定义（从 tools 模块导入）
```

## 3. 核心组件设计

### 3.1 AgentTool 类

封装单个工具的元数据和执行逻辑。

| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | str | 工具唯一标识名称 |
| `description` | str | 工具功能描述，供 LLM 理解 |
| `func` | Callable | 工具实际执行函数 |
| `parameters` | list[dict] | 参数列表（名称、类型） |

| 方法 | 返回 | 说明 |
|------|------|------|
| `run(**kwargs)` | str | 执行工具逻辑 |

### 3.2 @tool 装饰器

用于简化工具定义，自动从函数签名提取参数信息。

```python
@tool("工具名称", "工具功能描述")
def my_tool(param1: str, param2: int) -> str:
    """工具实现"""
    return "结果"
```

### 3.3 ToolCenter 类

工具管理中心，提供工具的注册、发现和查询功能。

| 方法 | 返回 | 说明 |
|------|------|------|
| `register(tool: AgentTool)` | None | 手动注册工具 |
| `unregister(tool_name: str)` | None | 注销工具 |
| `get_tool(name: str)` | AgentTool | 获取指定工具 |
| `list_tools()` | list[AgentTool] | 列出所有已注册工具 |
| `auto_discover(package: str)` | None | 自动扫描并注册工具目录下的所有工具 |
| `get_langchain_tools()` | list[Tool] | 转换为 LangChain 兼容的工具列表 |

## 4. 扩展机制

### 4.1 添加新工具

在 `src/graph_agent/tools/` 目录下创建新的 Python 文件（如 `web_tools.py`），使用 `@tool` 装饰器定义工具：

```python
from graph_agent.tools.base import tool

@tool("网页搜索", "搜索指定关键词的网页内容")
def web_search(keyword: str) -> str:
    """搜索实现"""
    return f"搜索结果: {keyword}"
```

### 4.2 自动发现

`ToolCenter.auto_discover()` 会自动扫描 `tools` 目录下所有非 `_` 开头的 `.py` 文件，注册所有 `AgentTool` 实例。

## 5. 与 LangGraph 集成

在 `graph.py` 中使用方式：

```python
from graph_agent.tools import ToolCenter

tool_center = ToolCenter()
tool_center.auto_discover()
tools = tool_center.get_langchain_tools()

# 绑定到 LLM
llm_with_tools = llm.bind_tools(tools)
```

## 6. 设计优势

1. **模块化**：工具按领域分类存放，便于维护
2. **可扩展**：添加新工具无需修改现有代码
3. **自动发现**：零配置即可使用新工具
4. **兼容性**：保持与 LangChain/LangGraph 生态兼容
5. **可测试**：工具可独立单元测试
