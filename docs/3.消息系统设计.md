# GraphAgent 消息系统设计文档

## 1. 概述

为了统一不同agent之间通信时使用的消息格式，方便实现后续的agent编排系统，GraphAgent 实现了统一的消息模块。

## 2. 目录结构

```
src/graph_agent/
├── message/
│   ├── __init__.py          # 公共接口导出
│   ├── base.py              # 核心基础类（ToolUseBlock/ToolResultBlock/ContentBlock/MessageBlock）
│   ├── convert.py           # 用于实现MessageBlock和langchain message之间的相互转换
│   ├── utils.py             # meaasge通用工具，例如为message生成id
│   └── message_type.py      # 详细定义了message_type的所有枚举值
└── graph.py                 # Agent 图定义（从 tools 模块导入）
```
